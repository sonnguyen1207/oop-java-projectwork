# 📚 School Library Management System

A JavaFX desktop application built with Java and Maven for Object-Oriented Programming coursework.

## 🚀 Features

* View library items (Books and Magazines)
* Borrow items
* Return items
* Different user types:

  * Guest
  * Student
  * Staff
* Borrow limits based on user type
* Item availability tracking
* JavaFX graphical user interface

## 🛠 Technologies Used

* Java 17
* JavaFX
* Maven
* Object-Oriented Programming principles
* MVC structure

## 📂 Project Structure

* `fi.oop.model.item` → Book, Magazine, LibraryItem
* `fi.oop.model.user` → Guest, Student, Staff, LibraryUser
* `fi.oop.service` → LibraryService
* `fi.oop` → MainController, App

## ▶️ Run the Project

```bash
mvn clean javafx:run
```

## 📖 Learning Goals

This project demonstrates:

* Encapsulation
* Inheritance
* Polymorphism
* Abstraction
* SOLID principles
* JavaFX UI development
